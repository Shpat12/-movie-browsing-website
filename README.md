# Movie Browsing Website

A full-stack movie browsing website with PHP backend and MySQL database, created during ROI Academy Web Development Advanced course.

## Features
- Movie catalog with database storage
- Browse movies by category
- Movie details pages
- PHP backend for data management
- MySQL database integration

## Technologies Used
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** PHP
- **Database:** MySQL

## About
This project was developed as part of the ROI Academy Web Development Advanced course (Nov 2023 - Mar 2024) to demonstrate full-stack development skills.

## Setup
1. Clone the repository
2. Import the database SQL file 
3. Configure database connection in PHP files
4. Run on local server (XAMPP, WAMP, etc.)
